# Concept
Titel: Peggle Exterminator!

Genre: Physics based classic arcade game 

# Beschrijving
De speler schiet een bal uit een kanon bovenaan het veld.
De gasbal stuitert van de insecten af.
Wanneer de bal de insecten raakt, verschijnt er een grafsteen, een dood insect, een plons, enz.
Wanneer de gasbal aan het einde in de prullenbak belandt, is de beurt voorbij.
Hoe meer insecten je uitroeit, hoe meer geld je krijgt en hoe hoger je geldmeter wordt.
Sommige insecten zijn meer waard dan andere.

# Gameplaykern
Bal: een bal van doodelijke gas

Pegs: insecten (kakkerlak, mug, vlieg, wesp, termiet)  

Score: punten per insect, Money Meter 

Doel: krijg je entire Money Meter om een bonus te halen! 

# Stijl van het spel 
- beetje vies
- spannend 
- geluiden van doode insecten, start/einde, etc

# Structuur
- Boven: Een exterminator kanon warmee de gas uitkomt
- Middel: de insecten die je punten kan scoren
- Onder: De vuilnesbak waar je kunt alle insecten weggooien
